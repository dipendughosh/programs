From the directory 'Important files' do the following:

Copy the files 'Msvcrtd.dll' & 'MFC42D.DLL' to 'WINDOWS\System32' directory or alternatively
     	run the 'instdll.exe' & follow the steps.
Copy the font 'AGENCYR.TTF' to 'WINDOWS\Fonts' directory or alternatively run 'AGENCYR.exe' &
	follow the steps.
Please read the document 'Calculator in Visual C++.doc' for operation details.

The executable 'calci_mfc.exe' can be found in the 'calci_mfc\debug' directory
